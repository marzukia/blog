A handful of tools built by Andryo Marzuki. Each one started as a problem I couldn't find a clean answer to.

---

## [colours](https://colours.mrzk.io)

A colour gradient and palette toolkit. Pick two colours, get back a smooth gradient with perceptually uniform steps and exportable swatches. Useful for design systems and data visualisation.

→ **[colours.mrzk.io](https://colours.mrzk.io)**

---

## [charted](https://charted.mrzk.io)

Zero-dependency SVG charting for Python. Pass in your data, get back a clean SVG — no matplotlib, no Plotly, no browser required. Works anywhere Python runs.

→ **[charted.mrzk.io](https://charted.mrzk.io)**
